En este primer cómputo, he aprendido mucho sobre desarrollo frontend. Aquí te comparto algunos de los temas más importantes.

Temas Aprendidos

Desarrollo Frontend con Librerías del Lado del Cliente

He aprendido a utilizar librerías como Bootstrap para agilizar el desarrollo y crear interfaces más atractivas y responsivas.

Bootstrap y JavaScript

Bootstrap no solo ofrece estilos CSS predefinidos, sino también componentes interactivos gracias a JavaScript. ¡Esto facilita la creación de elementos como menús desplegables y modales!

Estructuras y Funciones en JavaScript

JavaScript es el lenguaje que da vida a nuestras páginas web. He aprendido a organizar mi código utilizando estructuras de control (como bucles y condicionales) y funciones para reutilizar bloques de código.

Clases y Manipulación del DOM

¡La programación orientada a objetos también llegó al frontend! He aprendido a crear clases para representar elementos de mi página y a utilizar el DOM (Modelo de Objetos del Documento) para modificar el contenido y la estructura de mi HTML de forma dinámica.
